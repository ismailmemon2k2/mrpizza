````markdown
# MrPizza App

A modern pizza ordering mobile application built with **React Native and Expo**.  
MrPizza provides a user-friendly interface for browsing pizzas, exploring menu items, and managing orders through a smooth mobile experience.

## Tech Stack

- **React Native** - Mobile application framework
- **Expo** - Development platform and tooling for React Native
- **TypeScript** - Type-safe JavaScript development
- **Expo Router** - File-based navigation system
- **JavaScript / React Hooks** - Application logic and state management

## Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/ismailmemon2k2/mrpizza.git
````

### 2. Navigate to the project directory

```bash
cd mrpizza
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the Expo development server

```bash
npx expo start
```

After starting the server, you can run the application using:

* Expo Go (Android/iOS device)
* Android Emulator
* iOS Simulator
* Development Build

## Project Structure

```
mrpizza/

├── assets/           # Images, icons, and other resources
```

## Features

* Pizza menu browsing
* Order management
* Mobile-friendly user interface
* Fast development using Expo
* File-based navigation using Expo Router

## Development

Start editing files inside the `app` directory to build new screens and features.

To reset the default Expo template:

```bash
npm run reset-project
```

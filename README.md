# MrPizza App

A modern pizza ordering mobile application built with **React Native and Expo**.

MrPizza provides a user-friendly mobile experience for browsing pizzas, exploring menu items, and managing orders.

## Tech Stack

- React Native
- Expo
- TypeScript
- Expo Router
- JavaScript
- React Hooks

## Getting Started

Follow these steps to run the project locally.

### Clone the repository

```bash
git clone https://github.com/ismailmemon2k2/mrpizza.git
```

### Navigate to the project directory

```bash
cd mrpizza
```

### Install dependencies

```bash
npm install
```

### Start the Expo development server

```bash
npx expo start
```

You can run the application using:

- Expo Go (Android/iOS device)
- Android Emulator
- iOS Simulator
- Development Build

## Project Structure

```
mrpizza/

├── app/              # Application screens and routes
├── assets/           # Images, icons, and other resources
```

## Features

- Pizza menu browsing
- Order management
- Mobile-friendly user interface
- Expo-based development workflow
- File-based navigation using Expo Router

## Development

Edit files inside the `app` directory to add new screens and features.

To reset the default Expo template:

```bash
npm run reset-project
```
